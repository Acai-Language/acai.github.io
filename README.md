# Acai Official Website

This repository contains the source code for the official homepage of the **Acai** programming language. 

The live production site is deployed automatically via GitHub Pages and can be accessed at **[acai.org](https://acai.org)**.

## Repository Purpose
* Hosts the primary landing page, documentation links, and compiler download shortcuts.
* Managed directly by the core development team.

## Deployment Status
Every commit pushed directly to the `main` branch is instantly built and deployed to our live `.org` servers via GitHub Actions.
